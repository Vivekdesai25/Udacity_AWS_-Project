# AWS Static Website Hosting using Amazon S3 & CloudFront

## 📌 Project Overview

This project demonstrates how to deploy a **static website on AWS** using **Amazon S3 for storage** and **Amazon CloudFront as a Content Delivery Network (CDN)**.

The website is built using **HTML, CSS, and JavaScript** and delivered globally with low latency using CloudFront.

---

## 🚀 Technologies Used

* Amazon S3
* Amazon CloudFront
* HTML
* CSS
* JavaScript

---

## ⚙️ Deployment Steps

1. Created an **Amazon S3 bucket** to store the static website files.
2. Uploaded the website files including `index.html`, CSS, images, and vendor assets.
3. Enabled **Static Website Hosting** in the S3 bucket settings.
4. Configured a **Bucket Policy** to allow public read access to the website objects.
5. Created a **CloudFront Distribution** to serve the website through AWS CDN.
6. Set **index.html as the Default Root Object** in CloudFront.
7. Verified the deployment by accessing the website using the CloudFront domain.
